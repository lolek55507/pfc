This project is a simple javascript tictactoe Game

Where the player can play either against a friend

or against a bot that is build inside

  1. Position one is a button that when is clicked runs a bot the user can play against.
  2. Position two has a win counters for both circle and cross
  3. Position three has button that clears the scores
  4. Position four possesses two buttons one to clear the board and 2nd to generate a random move on the board
