This project is a simple rock-paper-scissors game

That has a button on the bottom of the page with inside text play

With a function that checks for user win and generates computer choice

  1. Section one is a wins counter that counts all users wins and substract the amount if the user loses
  2. Section two is a result section that display either the game ended as a draw or ended as win or lose
  3. Section three is a section that displays computer's choice and user choice
  4. Section 4 is users options that when each of them is pressed becomes a user choice
